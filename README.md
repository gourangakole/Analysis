# Analysis
   
### Download and compile the package   
    * git clone https://github.com/ravindkv/Analysis.git 

    * cd Analysis/src
    
    * make clean

    * make

    * cd .. 

### Compile and run the code
    * root -l 

    * .L hplusAnalyzer.C+

    * hplusAnalyzer t;

    * t.processEvents();


### Compile and run, in one go
    * root -l 'runme.C("hplusAnalyzer")'
